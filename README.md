Support
=========

A collection of widgets and utils that usually used in my projects.

##Usage

utils:
```groovy
compile 'com.liuguangqiang.support:utils:0.1.0'
```

widgets
```groovy
compile 'com.liuguangqiang.support:widgets:0.1.0'
```
